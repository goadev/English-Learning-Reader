# English Learning Reader

テキスト読み上げ機能を備えた英語学習支援 Web アプリケーション
geminiで生成コードをもとに作成した。

## 🎯 機能

- **テキスト読み上げ**: 入力されたテキストをブラウザの音声合成機能で読み上げ
- **再生速度調整**: 0.5x～2.0x の範囲で読み上げ速度をコントロール
- **ワンクリック操作**: 「再生」「停止」のシンプルなボタンインターフェース
- **グラスモーフィズムデザイン**: モダンで視認性の高いUI

## 📋 使い方

1. **テキストを入力**: テキストエリアに読み上げたいテキストを入力または貼り付け
2. **再生速度を設定**: スライダーで再生速度を調整（デフォルト: 1.0x）
3. **再生**: 「再生する」ボタンをクリックしてテキストを読み上げ開始
4. **停止**: 「停止」ボタンで読み上げを停止

## 🛠️ 技術スタック

- **HTML5**: Web Speech API を使用
- **TailwindCSS**: レスポンシブデザイン
- **JavaScript**: テキスト読み上げ制御ロジック

## 📝 ブラウザ対応

- Chrome / Edge（推奨）
- Firefox
- Safari

※ Web Speech API は最新ブラウザで対応しています

## 💡 使用例

- 英語テキストの音声確認
- 学習教材の読み上げ
- リーディングスキルの向上
